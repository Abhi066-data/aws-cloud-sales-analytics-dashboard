# Automated Sales Data ETL Dashboard System

## 📌 Project Overview

This project demonstrates a cloud-based ETL pipeline that automatically collects, processes, and stores sales data for analytics and dashboard reporting.

The system is built using AWS services and Python to automate the entire data pipeline from ingestion to visualization.

---

## 🚀 Project Architecture

Data Source → AWS S3 → AWS Lambda → RDS MySQL → Power BI Dashboard

---

## 🛠 Tech Stack

Python
AWS S3
AWS Lambda
AWS RDS (MySQL)
Power BI
SQL

---

## 📊 Features

* Automated ETL pipeline
* Serverless data processing
* Cloud storage for raw data
* Structured database storage
* Interactive Power BI dashboard

---

## 📈 Business Impact

* Reduced manual data processing
* Enabled real-time analytics
* Improved reporting efficiency
